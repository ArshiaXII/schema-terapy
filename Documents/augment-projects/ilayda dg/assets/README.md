- Yüksek çözünürlüklü fotoğraflar kullanın (en az 1920px genişlik)
- Mobil için optimize edilmiş boyutlarda olsun
- JPG formatında kaydedin
- Dosya boyutlarını makul tutun (her biri 500KB-2MB arası)
